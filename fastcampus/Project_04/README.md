## 폴더 구조

📁 src
├── 📁 components
│ │
│ ├── 📁 content # 페이지를 구성하는 컴포넌트 요소 관리 폴더
│ │ │
│ │ ├── 📁 canvas # 캔버스에 등장하는 모든 요소
│ │ │ │
│ │ │ ├── 📁 maps # 게임 맵 : 운동장, 마이룸, 미니게임 세가지 맵 구성
│ │ │ │ │
│ │ │ │ ├── 📁 player # 게임 캐릭터 모델링, 오브젝트
│ │ │ │ │ │ └── 📁 hooks # 게임 캐릭터 커스텀 훅
│ │ │ │ │
│ │ │ │ ├── 📁 structures # 각 맵에 속한 오브젝트 컴포넌트 총괄
│ │ │ │ │ │ └── 📁 ground # 운동장 맵
│ │ │ │ │ │ │ └── 📁 3dUIs # 운동장 맵의 Drei 기반 UI 컴포넌트
│ │ │ │ │ │ │ └── 📁 elements # 운동장 맵의 3D 오브젝트 요소들
│ │ │ │ │ │ │ └── 📁 npc # 운동장 맵의 NPC
│ │ │ │ │ │
│ │ │ │ │ └── 📁 miniGame # 미니게임 맵
│ │ │ │ │ │ │ └── 📁 elements # 미니게임 내 3D 오브젝트
│ │ │ │ │
│ │ │ │ └── 📁 myRoom # 마이룸 맵
│ │ │ │ │ │ └── 📁 elements # 마이룸 3D 오브젝트
│ │ │ │ │ │ └── 📁 placeMode # 마이룸 맵에서 배치시 사용되는 컴포넌트
│ │ │ │ │ │ └── 📁 rotationMode # 마이룸 맵에서 회전모드 시 사용되는 컴포넌트
│ │ │ │
│ │ ├── 📁 canvasLayout # 게임 내 캔버스 요소의 레이아웃 컴포넌트
│ │ │ │ │ └── 📁 canvasUserInterfaces # 일반 DOM UI 컴포넌트
│ │ │ │ │ │ └── 📁 common # 공통 UI
│ │ │ │ │ │ └── 📁 ground # 운동장 전용 UI
│ │ │ │ │ │ └── 📁 miniGame # 미니게임 전용 UI
│ │ │ │ │ │ └── 📁 myRoom # 마이룸 전용 UI
│ │ │ │
│ │ ├── 📁 loader # 로딩 화면 관련 컴포넌트
│ │ ├── 📁 lobby # 게임 입장 전 화면 컴포넌트
│
│
├── 📁 hooks # 커스텀 훅 관리
├── 📁 utilComponents # 클라이언트 사이드 소켓 이벤트 코드 관리
